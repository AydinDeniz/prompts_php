
# Social Media API Integration with PHP

## Requirements
- PHP >= 7.4
- API keys for Facebook, Twitter, and Instagram

## Setup
1. Replace placeholder API keys in `config/config.php` with real credentials.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` to fetch posts.

## Features
- Fetches latest posts from Facebook, Twitter, and Instagram.
- Displays posts dynamically using JavaScript.
- Requires authentication via API keys.
