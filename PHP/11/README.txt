
# AI Chatbot with PHP and OpenAI API

## Requirements
- PHP >= 7.4
- OpenAI API Key

## Setup
1. Replace `'your_openai_api_key_here'` in `config/config.php` with your actual OpenAI API key.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` in a browser to chat with the AI.

## Features
- Takes user input and sends it to OpenAI API.
- Displays real-time chatbot responses.
- Simple, lightweight frontend.
