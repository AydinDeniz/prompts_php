
# Advanced PHP-Based Parking Reservation and Management System

## Features
- Allows users to check available parking slots.
- Enables users to reserve parking spots.
- Simple UI for real-time parking availability.

## Setup
1. Create the `parking_slots` table using `create_tables.sql`.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` to manage parking reservations.

## Customization
- Integrate IoT sensors to update slot status automatically.
- Add payment processing for paid parking.
