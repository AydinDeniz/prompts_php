
# AI-Powered Image Recognition with PHP and Python

## Features
- Upload an image via a PHP interface.
- Python processes the image using a pre-trained AI model.
- Returns the predicted class from the AI model.

## Setup
1. Install Python dependencies: `pip install torch torchvision pillow`.
2. Run a local PHP server (`php -S localhost:8000 -t .`).
3. Open `http://localhost:8000/index.html` to upload images.

## Customization
- Modify `recognize.py` to use a different AI model.
- Improve the frontend design.
