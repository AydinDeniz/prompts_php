
# PHP Dynamic Form Builder with Validation

## Features
- Users can dynamically create form fields (text, email, number).
- Form validation (required fields, email validation).
- Form submission via AJAX.

## Setup
1. Run a local PHP server (`php -S localhost:8000 -t .`).
2. Open `http://localhost:8000/index.html`.
3. Add fields, fill in values, and submit the form.

## Customization
- Extend validation in `handle_form.php`.
- Add more input types in JavaScript.
