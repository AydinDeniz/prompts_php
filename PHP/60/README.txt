
# AI-Powered PHP Real Estate Listing and Valuation System

## Features
- Users can add and browse real estate listings.
- AI-powered property price estimation.
- Simple UI for property management.

## Setup
1. Create the `listings` table using `create_tables.sql`.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` to manage properties.

## Customization
- Add image uploads for listings.
- Integrate a map API for better location visibility.
