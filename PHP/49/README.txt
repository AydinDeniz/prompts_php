
# AI-Powered Personalized E-Commerce System

## Features
- AI-driven product recommendations based on user behavior.
- Randomized AI model for simulating personalized suggestions.
- Simple frontend interface for fetching product recommendations.

## Setup
1. Install dependencies if needed: `pip install numpy pandas`.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` and enter a User ID to get recommendations.

## Customization
- Connect to a real recommendation system using AI/ML models.
- Implement a user behavior tracking system for improved recommendations.
