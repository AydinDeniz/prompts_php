
# PDF Generation with PHP and mPDF

## Requirements
- PHP >= 7.4
- Composer installed

## Setup
1. Run `composer require mpdf/mpdf` in the project directory to install mPDF.
2. Ensure the `pdfs` directory exists and is writable.

## Usage
1. Open `generate_pdf.php` in a browser to use the HTML form.
2. Fill out the title and content to generate a PDF.
3. The generated PDF can be downloaded via the link after generation.
