
# PHP-Based Multi-Tenant Learning Management System (LMS) with Gamification

## Features
- Users can enroll in courses and earn points.
- Leaderboard system ranks top students.
- Simple front-end UI to manage courses and users.

## Setup
1. Create the tables using `create_tables.sql`.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` to enroll and view leaderboard.

## Customization
- Add quizzes to reward more points.
- Implement badges for achievements.
