
# PHP-Based Online Healthcare Consultation and Appointment System

## Features
- Patients can book appointments online with doctors.
- Displays upcoming appointments in a simple dashboard.
- Easy-to-use front-end interface.

## Setup
1. Create the `appointments` table using `create_tables.sql`.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` to book and view appointments.

## Customization
- Add video consultation functionality.
- Implement doctor availability schedules.
