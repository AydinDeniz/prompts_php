
# Real-Time AI-Driven Stock Market Prediction System

## Features
- Predicts stock market changes based on AI models.
- Simulated AI prediction logic for stock movement.
- Simple frontend for entering stock ticker symbols.

## Setup
1. Install dependencies if needed: `pip install numpy pandas`.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` to get stock predictions.

## Customization
- Connect to a live stock market API for real-time data.
- Integrate AI models for better prediction accuracy.
