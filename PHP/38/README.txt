
# Cryptocurrency Wallet and Transaction Management System

## Features
- Generates unique crypto wallets for users.
- Handles transactions between wallets.
- Simulates a simple blockchain-style balance update.

## Setup
1. Create the `wallets` and `transactions` tables using `create_tables.sql`.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` to create wallets and send transactions.

## Customization
- Integrate with real blockchain APIs for live transactions.
- Add user authentication for security.
