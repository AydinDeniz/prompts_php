
# PHP Event Ticket Booking System

## Features
- Lists available events from the database.
- Allows users to book tickets online.
- Stores bookings in a database.

## Setup
1. Create the `events` and `bookings` tables using `create_tables.sql`.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` and book tickets.

## Adding Events
- Insert events manually into the `events` table using SQL or a database tool.
