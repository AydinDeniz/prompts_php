
# AI-Driven Resume Parser and Job Matching System

## Features
- Extracts text from resumes using AI and NLP.
- Matches skills against job requirements.
- Provides job recommendations.

## Setup
1. Install dependencies: `pip install pdfminer.six`.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` to upload resumes.

## Customization
- Expand `parse_resume.py` to analyze job experience and education.
- Integrate a job board API for live job matching.
