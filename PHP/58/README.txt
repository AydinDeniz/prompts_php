
# PHP-Based Crowdfunding Platform for Non-Profit Organizations

## Features
- Users can create fundraising campaigns.
- Donations are tracked and updated in real-time.
- Simple UI to manage campaigns and contributions.

## Setup
1. Create the tables using `create_tables.sql`.
2. Run a local PHP server (`php -S localhost:8000 -t client/`).
3. Open `http://localhost:8000/index.html` to create and manage campaigns.

## Customization
- Add payment gateway integration for real donations.
- Implement user authentication for campaign owners.
