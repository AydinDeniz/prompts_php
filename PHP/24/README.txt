
# PHP Real-Time Cryptocurrency Price Tracker

## Features
- Fetches real-time cryptocurrency prices from the CoinGecko API.
- Displays live prices for Bitcoin, Ethereum, and Cardano.
- Updates prices every 30 seconds.

## Setup
1. Run a local PHP server (`php -S localhost:8000 -t .`).
2. Open `http://localhost:8000/index.html`.
3. Click "Refresh Prices" to fetch live data.

## Customization
- Modify `fetch_prices.php` to include more cryptocurrencies.
- Adjust the refresh interval in the JavaScript.
